◆ Input

enum Input : uint8_t

Defines bitmasks for player actions.

Each action is assigned a unique bit to pack multiple inputs into a single byte, saving network bandwidth. The combined state is sent in the inputs field of the PlayerInputPacket.

Valeurs énumérées
UP 

Move Up (Bit 0)

DOWN 

Move Down (Bit 1)

LEFT 

Move Left (Bit 2)

RIGHT 

Move Right (Bit 3)

SHOOT 

Shoot (Bit 4)
